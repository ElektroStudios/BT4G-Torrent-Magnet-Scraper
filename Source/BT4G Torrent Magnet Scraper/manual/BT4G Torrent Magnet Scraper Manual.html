01 · Overview

What is BT4G Torrent Magnet Scraper?

BT4G Torrent Magnet Scraper is a Windows desktop application that automates the process of finding torrents on the BT4G index website and collecting their magnet links — all without manual searching.

You define what to look for, what to keep, and what to throw away through three plain-text configuration files. The app does the heavy lifting: it queries BT4G, filters the results according to your rules, and delivers a clean list of magnet links ready to paste into your torrent client.

ℹ️ BT4G is a public BitTorrent DHT search engine. The scraper reads publicly accessible search result pages and extracts magnet link data. No account or login is required.
02 · How It Works

Processing Pipeline

Understanding the internal flow helps you configure the tool correctly:

input Search Terms.txt
→
http query BT4G.com
→
filter ① Forbidden Keywords
→
filter ② Required Keywords
→
extract Magnet Links
→
output Results

The two filters are applied sequentially: forbidden keywords are checked first — if any match, the torrent is immediately discarded. Only surviving results then go through the required-keywords check.

03 · Quick Start

Getting Up & Running

01
Locate the three configuration files
The configuration files are located inside the config folder, which sits in the same directory as BT4G Torrent Magnet Scraper.exe. Inside that folder you will find Search Terms.txt, Forbidden Keywords.txt, and Required Keywords.txt. Open them with any plain-text editor (Notepad, VS Code, Notepad++, etc.).
02
Fill in your search terms
Edit Search Terms.txt. Add one search term per line. These terms are passed directly to BT4G's search engine, so use the same wording you would type in a browser.
03
Define what to exclude (optional)
Edit Forbidden Keywords.txt. Add one keyword per line. Any torrent whose name contains any of these keywords will be silently discarded, no matter what. Great for filtering out unwanted formats, languages, sources, or qualities.
04
Define what to keep (mandatory)
Edit Required Keywords.txt. Add one keyword per line. A torrent is only collected if its name contains at least one of these keywords — this filter is always active and must contain at least one entry.
05
Run the application
Double-click BT4G Torrent Magnet Scraper.exe. The app will process each search term in sequence, apply your filters, and save the results to the output folder located next to the executable.
06
Check the output folder
Open the output folder once the app finishes. You will find one result file per search term — check its contents to find your collected magnet links.

The following table shows every possible combination of filter states and the outcome for a given torrent:

YES Any SKIP — Always discarded
NO YES PROCESS — Magnet link saved to output

06 · Best Practices

Tips for Great Results

🎯 Be specific with search terms. Instead of movie, use Inception 2010 BluRay. BT4G returns results ranked by relevance; specific terms give you cleaner, more manageable result sets.
🗂️ Use the forbidden list aggressively. Add cam rips, dubbed versions, unwanted languages, and known spam-release-group names. This is cheaper (computationally and mentally) than curating the required list to compensate.
🔤 Keyword matching is partial. A forbidden keyword of CAM will match any torrent name containing the string "CAM" — e.g. MovieName.CAM.rip.mkv. Keep keywords short and distinct to avoid unintended collateral filtering.
🪄 Want to accept virtually every search result? Add a single space ( ), a period (.), and a plus sign (+) to Required Keywords.txt (remember, one term per line!). Since almost every torrent name contains one of these characters, it acts as a near-universal match — handy when you only want the forbidden list to do the filtering.
07 · FAQ

Frequently Asked Questions

The results file is empty. What happened? ▼
Two likely causes: (1) Your search terms returned no results on BT4G — try simpler, broader terms. (2) Your filter combination is too restrictive — every result is being blocked by forbidden keywords, or none match your required keywords. Try temporarily clearing both filter files to confirm the scraper is working, then re-add filters gradually.
Does keyword matching care about uppercase/lowercase? ▼
Matching is case-insensitive. A forbidden keyword of cam will catch CAM, Cam, and cam alike. Write keywords however is natural for you.
Can I have multiple search terms on one line? ▼
No. Each line is treated as one complete search query. If you put multiple terms on a single line (e.g. Dune Blade Runner), BT4G will receive that entire string as a single query — which may or may not return what you expect. One term per line is the correct format.
I want to collect everything — how do I make Required Keywords match all results? ▼
Add a single space ( ), a period (.), or a plus sign (+) as an entry in Required Keywords.txt. Because virtually every torrent name contains at least one of these characters, this acts as a universal match — meaning only the forbidden list will do any filtering.
Can I comment out entries without deleting them? ▼
Yes! Prefix any line with # to turn it into a comment. The app will ignore it completely. This is useful for temporarily disabling a keyword or search term without losing it.

06 · Buenas Prácticas

Consejos para Mejores Resultados

🎯 Sé específico con los términos de búsqueda. En lugar de Inception, usa Inception 2010 BluRay. BT4G ordena los resultados por relevancia; los términos concretos generan conjuntos de resultados más precisos.
🗂️ Usa la lista de prohibiciones sin piedad. Añade cam-rips, versiones dobladas, idiomas no deseados y grupos de lanzamiento conocidos por spam. Es más eficiente que intentar compensarlo con la lista de requeridas.
🔤 La comparación es parcial. Una palabra prohibida "CAM" coincidirá con cualquier nombre que contenga la cadena "CAM" — por ejemplo "El camino a la perdición (2002).mkv". Ppuedes añadir espacios en blanco y signos de puntuación para evitar evitar filtraciones no deseadas por colisión, por ejemplo " CAM " o ".CAM.".
🪄 ¿Quieres aceptar prácticamente todos los resultados? Añade un espacio en blanco ( ), un punto (.) y un signo más (+) en Required Keywords.txt (recuerda, ¡un término por línea!). Como casi todos los nombres de torrents contienen alguno de estos caracteres, actúa como un comodín universal — perfecto cuando solo quieres que sea la lista prohibida quien filtre.
07 · Preguntas Frecuentes

FAQ

El archivo de resultados está vacío. ¿Qué ha pasado? ▼
Dos causas probables: (1) Tus términos de búsqueda no devuelven resultados en BT4G — prueba con términos más simples y genéricos. (2) Tu combinación de filtros es demasiado restrictiva — todos los resultados son bloqueados por la lista prohibida o ninguno coincide con las palabras requeridas. Vacía temporalmente ambos archivos de filtros para confirmar que el scraper funciona, y luego vuelve a añadir los filtros poco a poco.
¿La comparación de palabras distingue mayúsculas de minúsculas? ▼
La comparación es insensible a mayúsculas/minúsculas. Una palabra prohibida cam detectará CAM, Cam y cam por igual. Escribe las palabras clave como te resulte natural.
¿Puedo poner varios términos de búsqueda en una sola línea? ▼
No. Cada línea se trata como una consulta de búsqueda completa. Si pones varios términos en una línea (p. ej. Dune Blade Runner), BT4G recibirá esa cadena completa como una única consulta, lo que puede no devolver lo que esperas. Una línea = un término: es el formato correcto.
Quiero recopilar todo, ¿cómo hago que Palabras Requeridas coincida con todo? ▼
Añade un espacio en blanco ( ), un punto (.) o un signo más (+) como entrada en Required Keywords.txt. Como prácticamente todos los nombres de torrents contienen alguno de esos caracteres, actúa como comodín universal — solo la lista prohibida filtrará resultados.
¿Puedo comentar entradas sin borrarlas? ▼
¡Por supuesto! Pon # al inicio de cualquier línea para convertirla en comentario. La aplicación la ignorará completamente. Muy útil para deshabilitar temporalmente una palabra clave o un término de búsqueda sin perderlo.
